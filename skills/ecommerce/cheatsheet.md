# 电商素材速查（创作时必读）

> 顺序：**确认闸门** → 本文件定工作流 → 按垂类打开 **1 个** reference。闸门见 [workflows/confirmation-gates.md](workflows/confirmation-gates.md)。

## 标准链路

```
产品输入（白底/实拍/链接/竞品）
  → 场景/模特/氛围（生图）
  → 详情页 / 主图 / 海报（平面）
  → 图生视频（种草 / 口播 / 变装）
  → 剪辑包装 → 批量裂变（多场景/比例/风格）
```

## 各平台要点

| 平台 | 画幅 | 节奏 | ❌ 避免 |
|------|------|------|--------|
| 淘宝/天猫详情 | 790 宽长图 | 卖点图标化、科技微距 | 纯氛围无参数 |
| 亚马逊头图 | 1:1 / 白底规范 | 信息直给、对比图 | 过度滤镜 |
| TikTok / 短视频 | 9:16 | 前 3 秒钩子、UGC 感 | 广告腔、假模特 |
| 小红书 / Ins | 3:4 / 1:1 | Plog、平铺、氛围 | 硬广字幕堆满 |

## 输入素材决策

| 你有什么 | 优先路径 |
|---------|---------|
| 仅白底图 | 场景裂变 → 主图/详情 → 图生视频 |
| 实拍 + 白底 | 实拍做锚点，AI 扩场景 |
| 仅商品链接 | Keevx 等链接导入 + 反推风格再生成 |
| 竞品详情页 | 先 AI 拆解结构再复刻（见 workflows） |

## 一致性锚点（电商必守）

| 锚点 | 写法 |
|------|------|
| 产品外观 | 固定参考图 +「保持产品形状/Logo/配色不变」 |
| 同一模特 | 定妆照/三视图锁脸 + 每镜同参考 |
| 多套穿搭 | 先锁人再换衣，或黑衣人/赛博试衣工作流 |
