# 产品 → 电商素材流水线

> 闸门收齐后执行。提示词技法 → `prompt-director`；品牌 TVC 质感 → `brand-ad-cg`。

## 模式选择

| 模式 | 触发 | 核心 reference |
|------|------|----------------|
| 详情页 | `detail_page` | workflows.md + fashion/3c |
| 主图裂变 | `main_image` + 白底 | workflows.md |
| UGC 种草 | `ugc_video` | fashion.md |
| 图生视频铺量 | `product_video` + 裂变 | fashion.md / 3c-digital.md |

## 详情页四阶段（通用）

```
1. 竞品/参考拆解（结构+视觉密码，非直接抄图）
2. 定屏结构（首屏符号 → 卖点 → 微距 → 场景 → 转化）
3. 逐屏生成 + 文字/Logo 修正
4. 文案赋能 + 长图拼接
```

## 白底裂变三阶段

```
1. 锁产品（参考图 + 不变项）
2. 场景/模特/氛围批量（多 prompt 或多工具并行）
3. 统一调色 + 比例导出
```

## UGC 视频三阶段

```
1. 定「人设脚本」（对镜拍/口播/展示）
2. 首帧真人感（拒绝假人 → 见 fashion.md）
3. 图生视频 + 剪映包装（字幕/BGM/节奏）
```
