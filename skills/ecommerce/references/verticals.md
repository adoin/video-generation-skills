## 保健品（Urdru 系列）

### 多语种口播

| 步骤 | 工具 | 要点 |
|------|------|------|
| 模特生图 | Midjourney | 正脸/光照均匀/表情亲切；公式：主体+场景动作+真实抓拍风格 |
| 产品换物 | SD 换物工作流 | 蒙版替换手中产品为真实 SKU |
| 微动视频 | MJ 图生视频 | 头部微转、嘴部动作，为对口型做准备 |
| 对口型 | 后续口播管线 | 脸正对镜头，无遮挡 |

**模特关键词结构：**
- 主体：年龄+人种+发型+服装
- 场景：晨光厨房/窗边等明亮环境
- 风格：candid photography / natural morning light / high resolution

### 虚拟主播带货

规避真人拍摄成本与模特版权风险。

```
MJ 虚拟模特(正脸无遮挡) → SD FusionX-Phantom 插件 + 产品图 → 带货视频 → 接入多语种口播
```

### 功效可视化视频

| 视频类型 | 静帧要点 | 动态要点 |
|----------|----------|----------|
| 口服→体内 | 半透明人体+器官，饮用罐装产品 | ComfyUI 图生视频，器官由暗变正常 |
| 肝脏排毒 | 发光粒子作用于肝脏 | 脏污/毒素动态脱落 |
| 消化能量 | 暖橙红光芒包裹消化道 | 能量持续流动 |

**即梦生图关键词：** translucent body / internal organs / glowing particles / warm orange energy flow。

### 身材延时摄影

| 步骤 | 动作 |
|------|------|
| 初始形象 | 肥胖中年男性全身正面，光照均匀 |
| 瘦身对比 | SD 变瘦工作流生成瘦版 |
| 首尾帧视频 | 即梦 A→B 帧，多生成几段变体 |
| 序列帧合成 | 导出帧 → 延时摄影节奏剪辑 |

### 品牌广告（都市男性精力叙事）

| 要素 | 设定 |
|------|------|
| 诉求 | 疲倦→服用→活力提升 |
| 主人公 | 28-35 岁都市白领，服用前后眼神对比 |
| 场景 | 晨起/早饭/办公/健身/家庭，6 镜 25s 内 |
| 流程 | 分镜脚本 → 首帧+视频双提示词 → 即梦/Seedance 生成 → 剪映 |

### ❌ 不要

- 夸张动作或侧脸（对口型失败）
- 功效表述违反广告法规（教程仅讲视觉技法）

### ✅ 要

- 正脸、均匀光、手持产品无遮挡
- 半透明/能量流等「可视化」降低理解成本

## 美妆

### 情人节营销大片（FlowPix）

三方案并行架构：

| 模块 | 作用 |
|------|------|
| 输入区 | 产品实拍 + 卖点文案 + 自定义风格词 |

| 执行 LLM | 方案 → 逐条绘画提示词 |
| 并发出图 | 3 条线同时生成，一次拿 3 张海报 |

### 美妆 UGC 视频

首尾帧接龙解决长视频脸崩：

| 步骤 | 要点 |
|------|------|
| 黄金首帧 | UGC style / phone camera / messy background |
| 关键帧序列 | Nano Banana 按脚本生成连贯帧 |
| 逐段视频 | Veo 3.1 首尾帧接龙 |
| 拼合 | 剪映，保持手持晃动质感 |

## 家居

### 宜家级商业宣传图

| 步骤 | 工具 | 要点 |
|------|------|------|
| 高级背景 | Midjourney | 只造环境不造产品；editorial minimal / 单侧柔光 |
| 产品植入 | 即梦 | 上传官方产品图融合场景 |
| 一致性 | 固定光位+色温+材质描述 | 多镜头同套提示词骨架 |
| 视频化 | 即梦图生视频 | 慢推镜头展示空间 |

**提示词拆解维度：** 主体卖点 / 极简背景 / 编辑风 / 构图光位 / 低饱和色彩。

### 跨境家具场景图（Nano Banana）

| 步骤 | 动作 |
|------|------|
| 素材 | 纯白底家具图 + 居家风 AI 模特（半靠沙发等自然动作） |
| 融合生成 | 产品+模特+场景提示词 |
| 比例适配 | 1:1 主图 / 3:4 详情 / 9:16 A+ 页，即梦裁切扩展 |
| 质检 | 家具不变形、占画面 70-80%、缝线木纹清晰 |

### ❌ 不要

- 北欧沙发配中式红木
- 场景杂物抢家具注意力

### ✅ 要

- 纯色底产品图便于 AI 识别轮廓
- 模特动作用「使用场景」而非摆拍凝视

## 品牌提升 · Nano Banana 电商

### AI 模特诞生（3C 耳机案例）

| 步骤 | 动作 |
|------|------|
| 参考图 | 收集大厂耳机宣传图，MJ 反推并差异化修改 |
| 模特生成 | Midjourney 固定光影与姿势 |
| 产品渲染 | Blender 按模特光位渲染耳机 3D 模型 |
| 合成 | SD 溶图/融合工作流 |

### AI 换衣服

1. 选一张满意模特底片（或按上述流程生成）
2. 上传新款平铺服装图
3. Nano Banana 自然语言换装

### AI 修图

一键救活灰蒙产品图：对比度/张力/高级感调色，无需精通 PS。

### 跨境场景适配

| 输入 | 动作 |
|------|------|
| 产品白底图 | 高分辨率干净背景 |
| 目标市场信息 | 地域文化/典型生活场景 |
| 风格参考 | 当地室内设计图 |

**多市场 Prompt 要素：** 地域建筑元素 + 文化符号 + 情感氛围（精油等情感消费品加重氛围词）。

示例场景：法国高端沙龙 / 日本禅意家居 / 泰国夜市街头。

## 食品 · 膨化零食宣传

| 步骤 | 动作 |
|------|------|
| 模特生成 | DeepSeek 写 Prompt → MJ 生成；**优先保证面部自然** |
| 面部精修 | SD 电商场景修图工作流，去涂抹感 |
| 产品替换 | SD 换物，手持虾条包装 |
| 场景海报 | 按品牌调性生成多版宣传图 |

### ❌ 不要

- 忽视 MJ 手指变形（后期必修）
- 面部有问题仍继续合成（修脸成本远高于重生成）

### ✅ 要

- 表情自然优先，手部可后期修
- 分步：高清修复 → 换产品 → 场景合成

## 塑身衣 · TikTok 出海（Uncolive）

静态模特照 → 动态东南亚带货视频。

| 步骤 | 工具 | 要点 |
|------|------|------|
| 氛围参照 | Sora 2 | `Southeast Asia shapewear` + `10-second TV advertisement` 锁定时长 |
| 分镜锁定 | Midjourney | 客户模特照垫图中高权重，保留面部 |
| 场景反推 | 从 Sora 画面反推 MJ 分镜提示词 |
| 剪辑合成 | 剪映 | 统一节奏与东南亚审美 |

**关键词技巧：** 加入 `TikTok style` / `TV advertisement` 提升生成质量。

## 垂类通用原则

| ❌ 不要 | ✅ 要 |
|--------|------|
| 功效/疗效违规表述 | 视觉叙事：使用前后面部状态、生活场景变化 |
| 单张产品图硬套全球场景 | 按市场单独写 Prompt 文化元素 |
| 模特太完美无瑕疵 | 适度真实感（尤其 UGC/买家秀类） |
| 跳过三视图/多角度 | 食品/保健品手持物必须看清包装 |
