# 电商通用工作流模式

> 涵盖：白底裂变 · 单品全套物料 · 出海铺量 · 视频四步解耦 · 宠物带货 · 亚马逊头图 · 链接导入成片 · TVC/UGC 复刻

---

## 模式总览

```
                    ┌─────────────────────────────────────┐
  输入层            │ 产品图 / 链接 / 对标视频 / 卖点文案  │
                    └──────────────┬──────────────────────┘
                                   ▼
                    ┌─────────────────────────────────────┐
  智能体层          │ LLM 策划 · 分镜 · 提示词 · 文案      │
                    └──────────────┬──────────────────────┘
                                   ▼
                    ┌─────────────────────────────────────┐
  生成层            │ 静帧 → 视频 → 多语言/多比例          │
                    └──────────────┬──────────────────────┘
                                   ▼
                    ┌─────────────────────────────────────┐
  交付层            │ 剪映合成 · 去水印 · 放大 · 排版       │
                    └─────────────────────────────────────┘
```

---

## 模式 A：白底图裂变

```
白底图 → LLM 产品分析 → 分镜策划 → 多风格静帧 → 图生视频 → 批量切片
```

| 变体 | 产出 |
|------|------|
| 10 条广告片 | 3 全景 + 7 微距分镜 |
| 4 生活方式 | 4 套 Look + 对镜视角 |
| 6 段种草 | Vlog 分镜 + 口播文案 |
| 9 张 Plog | 3-3-3 产品/情绪/氛围 |

### ❌ 不要

- 未确认静帧全开视频（积分浪费）
- 抽象卖点词直接进视频模型

### ✅ 要

- 材质/版型让 LLM 先分析再策划
- 先图后视频，分组执行

---

## 模式 B：单品 → 全套物料

```
单品图 → 详情页 + 主图 + 种草视频 + Plog/节日海报
```

| 节点 | 工具范式 |
|------|----------|
| 详情页 | 竞品拆解 → 10 屏结构 → 逐屏生成 |
| 主图 | 中焦+全身+微距 SHEIN 标准 |
| 节日 | 产品信息 + 风格偏好 → 灵感→提示词→海报 |
| 视频 | 同源白底图接入种草管线 |

---

## 模式 C：出海短视频铺量

```
参考图 → Sora2 竖屏短视频 → 批量变场景 → 去水印 → 2K 放大 → 矩阵投放
```

| 要素 | 写法 |
|------|------|
| 画幅 | 9:16 竖屏 |
| 时长 | Prompt 内写 `10-second` / 选 15s |
| 质感 | `fast handheld camera` / `busy street` |
| 批量 | 改场景词 10-20 条；FFMPEG/ComfyUI 批量后处理 |

## 模式 D：爆款视频裂变（四步解耦）

核心问题：超长提示词导致 AI 注意力衰减、分镜混乱。

**解法：** 四个智能体各司其职，任务解耦。

| 智能体 | 角色 | 输出 |
|--------|------|------|
| 1. 全息商品解析器 | 视觉降维 | 材质/光泽/色温等物理参数词典 |
| 2. 逆向工程师 | 坐标剥离 | 时间戳 + 运镜 + 旧商品空间位置 |
| 3. 规划导演 | 时空切块 | 按 10-15s 切 Unit，商品置换进原骨架 |
| 4. 提示词架构师 | 纯净编译 | 分镜表：镜头/时间线/文案/中文画面词 |

### 实操流程

| 步骤 | 动作 |
|------|------|
| 选品 | 多角度产品高清图 |
| 找对标 | Kalodata 等找高转化爆款视频 + 提取口播文案 |
| 智能体 1 | 商品图 → 视觉特征表 |
| 智能体 2 | 对标视频 → 运镜时间轴 |
| 智能体 3 | 特征+时间轴 → 分段方案 |
| 智能体 4 | 方案 → Seedance/Sora 施工表 |
| 生成 | 即梦按表逐 Unit 生成 |
| 合成 | 剪映按原节奏拼接 |

### ❌ 不要

- 商品介绍+分镜+运镜+文案塞进一段 Prompt
- 跳过对标视频逆向

### ✅ 要

- 明确「何时张嘴/何时闭嘴」
- 每段从 0 秒起算（适配分段生成逻辑）

## 模式 E：宠物带货视频（三步）

```
宠物图 + 商品图 + 环境图 → 提示词智能体 → Seedance 2.0 → 剪映
```

### 智能体迭代史（带货导演思维）

| 版本 | 问题 | 改进 |
|------|------|------|
| V1 | 静态描述，像壁纸 | — |
| V2 | 画面有人手人脚 | 颁布「人类禁令」；图1/2/3 精准打标 |
| V3 | 动作无力，宠物睡觉 | **8 秒时间轴**：0-2s 猛扑吸睛 → 2-6s 微距把玩 → 6-8s 甩动收尾 |

**动作词库：** 飞速跑入 / 猛扑 / 疯狂拨弄 / 啃咬 / 疯狂甩动 / 极速穿梭。

### 操作

1. 豆包/Gemini 创建「宠物视频提示词」智能体（经 V1→V3 调优）
2. 上传宠物+商品+环境三图 → 获分镜提示词
3. 即梦 Seedance 2.0 全能参考生成 → 剪映加字幕 BGM

## 模式 F：亚马逊头图十件套

```
多角度实拍 → 6视图锁定 → 风格参考+卖点 → 10张策划 → GPT Image 2.0 批量 → 多语言切换
```

| 步骤 | 动作 |
|------|------|
| 原材料 | 产品多角度图 + 风格参考图 + 卖点文案 |
| 6 视图 | ChatGPT GPT Image 2.0 生成标准 6 面视图，锁定产品外观 |
| 视觉策划 | 6 视图 + 风格参考 + 卖点 → LLM 规划 10 张图内容 |
| 批量出图 | 按策划逐张生成 |
| 多语言 | 同一视觉结构，切换日/德/法等文案层 |

### 策划提示词原理

LLM 扮演「视觉策划」：每张图对应一个买家决策点（卖点可视化/场景化/对比/尺寸参照…）。

### ❌ 不要

- 每张图独立生成不管产品一致性
- 跳过 6 视图直接出头图

### ✅ 要

- 6 视图作为全批次「产品锚点」
- 风格参考图统一色调与排版逻辑

## 模式 G：链接/URL 一键成片

```
亚马逊链接 → 提取商品信息 → 选数字人+音色 → 生成多版带货视频
```

扩展能力：100+ 语言翻译 + 口型同步。

适用于：快速测款、多语种铺量，画质要求低于定制工作流时。

---

## 模式 H：视频复刻（TVC / UGC）

```
对标视频 → 镜头分割 → 首帧长图 → 产品替换 → 逐镜生成 → 节奏拼合
```

| 类型 | 侧重点 |
|------|--------|
| TVC | 产品特写、光影、品牌调性；GPT Image 2.0 替换产品 |
| UGC | 手持晃动、生活场景、不完美构图 |

## 交付质检清单

| 检查项 | 标准 |
|--------|------|
| 分辨率 | 1080p+ 商业投放 |
| 水印 | 已去除 |
| 产品一致性 | 6 视图/锚点图对齐 |
| 字幕 | AI 画面无字，后期单独加 |
| 口型 | 多语种需口型同步检查 |
| 比例 | 平台规格（9:16 / 1:1 / 3:4） |
| 批量 | 脚本化后处理，非逐个手工 |

---

## 避坑总则

| ❌ 不要 | ✅ 要 |
|--------|------|
| 大锅炖超长 Prompt | 智能体/分步解耦 |
| 跳过竞品/对标分析 | 先拆再产 |
| 生成完直接投放 | 放大+去水印+多版本 A/B |
| 整组执行不审核 | 逐节点验收 |
| 忽视平台合规 | 功效/对比/夸大表述人工终审 |
