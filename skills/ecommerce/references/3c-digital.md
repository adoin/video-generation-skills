## 漫步者音箱系列（详情页 + 视频 + 切片）

同一产品线的三条产线，工具链可复用。

### 双智能体详情页（豆包 + 即梦）

| 角色 | 职责 |
|------|------|
| 策划智能体 | 竞品分析 → 卖点框架 → 分模块文案 |
| 提示词智能体 | 文案 → 逐屏生图提示词 + 风格定调 |

**出图策略：** 即梦图片 3.1，尺寸 512×2016，**2 屏一组**批量生成 → Figma 排版。

### 详情页 3.0（豆包/Gemini + Lovart + GPT Image 2.0）

| 步骤 | 动作 |
|------|------|
| 素材准备 | 对标详情页长图 + 产品图 + 参数表 |
| 策划智能体 | 分步投喂：对标图 → 产品图 → 参数 → 生成指令 → 10 大模块文案 |
| Lovart 落地 | 素材拖画布，Agent 对话生成；**先出 5 屏测试 Apple 风**，满意再续后 5 屏 |
| 高阶微调 | 局部重绘、文字编辑、风格统一 |

### 产品宣传视频（即梦 Agent + 哩布 ViduQ2）

| 步骤 | 工具 | 要点 |
|------|------|------|
| 可落地分镜 | 即梦 Agent 模式 | 上传产品图，复制 Agent 输出分镜 |
| 提示词精炼 | 豆包 | 去多余元素（波纹/手绘质感），适配场景 |
| 视频生成 | Liblib ViduQ2 | 按分镜逐段生成 |
| 合成 | 剪映 | 拼接 + BGM |

### Sora2 日更切片

| 步骤 | 动作 |
|------|------|
| 转化型提示词 | 豆包生成 10+ 切片脚本（卖点/场景/时长） |
| Storyboard | Sora2 逐分镜上传音箱参考图，对齐脚本时长 |
| 剪映 | 批量导入，节奏剪辑 |

**产能参考：** 单账号日更 5-8 条，转化率较传统制作提升约 30%。

### ❌ 不要

- 换产品时不重建智能体策划（提示词会崩）
- 单屏超长提示词塞全部卖点

### ✅ 要

- 竞品图 + 自有产品图分步投喂
- 2 屏一组迭代，先文案后视觉

## 产品海报

### 卖点海报（Gemini Gems + Nano Banana Pro）

| 步骤 | 输入 |
|------|------|
| 创建智能体 | 「产品卖点海报策划」 |
| 四件套投喂 | 参考海报 + 产品图 + 核心卖点文字 + 氛围关键词 |
| 出图 | Nano Banana Pro 生成**无字版底图** |
| 排版 | Figma/PS 叠加文案 |

卖点写法示例：功能点拆条（吸力/控水/防缠毛/自清洁…），避免抽象形容词。

### 大透视海报（FlowPix 工作流）

| 步骤 | 要点 |
|------|------|
| 捏脸换装 | 文本定义模特 + 上传穿搭参考，融合固定形象 |
| 产品三视图 | **正/侧/背**多角度上传，防大透视变形 |
| 场景生成 | LLM 策划 2 种高级场景 × 10 种动作 |
| 批量输出 | 纯平视透视原则，产品占视觉重心 |

### 逆光剪影美学

| 步骤 | 工具 |
|------|------|
| 艺术总监 Gem | Gemini/豆包创建「3C 视觉艺术总监」智能体 |
| 需求输入 | 产品名 + 聚焦部位（如 AirPods 塞与腔体） |
| 方案输出 | 3 种光影排版方案 |
| 生图 | Nano Banana Pro 按方案执行 |
| 排版 | Figma 叠加标题与发布信息 |

**光影关键词：** backlit silhouette / rim light / minimal mystery / product hero framing。

### 多产品组合场景（小米系 Banner）

四引擎架构：

| 引擎 | 功能 |
|------|------|
| 输入端 | 多产品抠图 + 风格参考图，**标注尺寸** |
| 策划大脑 | 纯平视透视，多套几何展台方案 |
| 环境生成 | 水磨石/石膏/木材质感 3D 展台 |
| 合成端 | 参数化比例锚定，统一全局光影 |

自定义文本示例：`待放入主体` + `期望氛围` + `主色调`。

### ❌ 不要

- PS 硬拼多产品（透视穿帮）
- 省略产品尺寸标注

### ✅ 要

- 选策划方案后再跑环境+合成
- 商品占画面 70%-80%

## 产品 CG 视频（Seedance 2.0）

### Gemini Gems 提示词工厂

| 步骤 | 动作 |
|------|------|
| 提取公式 | 从优质 15s 产品大片反推提示词结构（豆包提取） |
| 逆向工程 | Gemini 拆解为可复用模板 |
| 创建 Gem | 「Seedance 分镜提示词生成器」 |
| 试用调优 | 生成 15s 词 → Seedance 实测 → 反复推敲 |
| 量产 | 输入产品图 → 自动输出分镜 → 即梦 Seedance 生成 |

### 挂脖风扇广告（五步工作流）

| 步骤 | 工具 | 动作 |
|------|------|------|
| 剧本设定 | Gemini | 多方案创意 → 选定 → 细化人物环境 |
| 场景角色图 | GPT Image 2.0 | 产品图+设定文案，附反向提示词保画质 |
| 主角真人化 | GPT Image 2.0 | 反复微调至自然写实 |
| 分镜生成 | GPT Image 2.0 | 按剧本逐镜，保持角色一致 |
| 动态合成 | 即梦 Seedance 2.0 | 分镜图生视频 → 剪映 |

### 泰式反转广告

| 步骤 | 动作 |
|------|------|
| 剧本策划 | Gemini 构建冲突+反转剧情框架 |
| 细化设定 | 选定剧本 → 角色/场景/道具细化 |
| 超写实场景 | GPT Image 2.0 + 前置/后置/反向提示词 |
| 人物三视图 | 建立角色资产包 |
| 分镜+视频 | 按镜生成 → Seedance 动态 → 剪映 |

**核心：** 泰式广告 = 合理逻辑 + 极致冲突感反转。

## 品牌级产品图（魔音音箱）

### 光影重塑四步法

| 步骤 | 动作 |
|------|------|
| 解构产品图 | 识别硬光/百叶窗阴影 vs 目标柔和场景 |
| 造境 | Midjourney 单独生成卧室晨光场景（不含产品） |
| 移花接木 | 即梦 4.0 上传产品+背景，指令「光影替换」保留材质抛弃原光 |
| 后期 | 检查接触阴影与色温一致 |

### ❌ 不要

- 直接生成带产品的场景（环境控制力弱）
- 简单抠图粘贴（光影不匹配）

### ✅ 要

- 先造境再入戏
- 强调 contact shadow / warm morning light / seamless integration

## 3C 通用提示词结构

```
主体/细节 → 环境/场景 → 风格 → 构图/镜头 → 色彩 → 渲染器/清晰度
```

| 要素 | 3C 写法示例 |
|------|-------------|
| 主体 | 产品形态、材质、按键/接口细节 |
| 环境 | 极简棚拍 / 生活场景 / 3D 展台 |
| 风格 | editorial minimal / Apple-grade / Octane commercial |
| 构图 | 中长焦 / 大透视仰拍 / 微距特写 |
| 色彩 | 中性低饱和，产品色为主角 |
| 真实感 | photo-realistic / ray tracing / studio lighting |

## 避坑清单

| ❌ 不要 | ✅ 要 |

| 营销黑话直接喂视频模型（「性价比」） | 翻译成视觉语言：金属拉丝、冷色调灯光 |

## 15s 产品 CG 广告：六视图 + 九宫格分镜（info-2903）

> 溯源：[info-2903](https://www.super-i.cn/info-2903.html)（2026-06 站更）
> 品牌 CG 同类 → `brand-ad-cg/references/product-cg.md`

**适用：** 鼠标、耳机、键盘、音箱、手表、香水等结构型 3C 展示短片。

### 五步流程

| 步 | 工具 | 产出 |
|----|------|------|
| 1 | — | 3–4 张多角度产品实拍/官图（正/侧/斜侧/顶） |
| 2 | GPT Image 2.0 | **六视图**（正/背/左/右/俯/底）锁外形 |
| 3 | GPT/Gemini | 15s 分镜脚本 + **九宫格分镜 prompt** |
| 4 | GPT Image 2.0 | 3×3 分镜设计图（六视图一并上传） |
| 5 | Seedance 2.0 | 六视图 + 九宫格 + 分镜脚本 → 生成 |

### 三类素材分工

| 素材 | 锁定 |
|------|------|
| 六视图 | 产品不变形、按键/Logo 不丢 |
| 九宫格分镜图 | 9 个关键画面顺序与构图 |
| 分镜脚本 | 运镜、节奏、时长、声音氛围 |

| ❌ 不要 | ✅ 要 |
|--------|------|
| 单张产品图直接做 15s 视频 | 先六视图再分镜 |
| 跳过脚本直接抽视频 | 先规划广告节奏再出九宫格 |
| 营销词喂视频模型 | 翻译成结构、材质、镜头语言 |
