## §42 局部元素融合

**本质：** 精准替换 = 分步提供光影材质参考，而非一句全局改写。

### 参照物定调法（画风差异大 / 独立运动）

### ❌ 不要


把火车换成红色托马斯火车

- 直接视频重绘 → AI 调取卡通训练集，无法脑补电影级光影，贴片感

### ✅ 要


1. 截取原视频首帧
2. Nano Banana Pro 局部重绘：在原光影下替换目标物体（托马斯火车融入暖调电影光）
3. 可灵/即梦多图参考：原视频 + 修正静帧，提示词「保持背景光影与镜头运动，火车与参考图一致」


### 静帧遮罩法（大角度运动 / 复杂受光）

### ❌ 不要

- 单张定调 + 物体侧转背面 → 中间帧融化变形

### ✅ 要


1. 提取 4+ 关键帧（侧视 / 中途 / 转向 / 背面）
2. 设计软件：玩具/替换物按各角度画蒙版
3. Nano Banana Pro：蒙版 + 参考物逐帧图生图定调
4. 可灵 3.0 Omini：视频 + 按时间顺序排列的修正帧，多图参考驱动

## §44 视觉标注与精准修正

**原理：** 图片与涂抹标记是**最高权重提示词**；文本方位词（左边/那个）命中率随画面复杂度断崖下跌。

### 局部替换

### ❌ 不要


删除画面左下角桌子旁边的那个红色杯子

- 两个相似物体或前后遮挡 → 盲猜像素区，连带改坏

### ✅ 要


1. 涂抹/框选需修改区域（Nano Banana / MJ Vary Region / 即梦局部重绘）
2. 文本只写目标内容，零方位词：vintage brass alarm clock
3. AI 锁定选区内像素重组，环境光影自动融合


### 空间位移

### ❌ 不要

- 重写整段提示词抽卡搬家 → 背景构图全毁

### ✅ 要


1. 原位置涂抹物体（保留轮廓边缘，勿盖住内容）
2. 目标位置打点/箭头标注
3. 提示词：move the marked object to the red arrow position, maintain scale and lighting


### 涂抹致命细节

### ❌ 不要

- 涂抹盖住挂画内容/物体关键特征 → AI 眼里是黑洞，重新乱猜

### ✅ 要

- 沿外轮廓涂抹，保留形状与材质信息
- 移动挂画：标边缘框，不覆盖画面内容

**思维切换：** 从「电话指挥瞎子」→「指着画布明确边界」= 工业化可控工作流。
