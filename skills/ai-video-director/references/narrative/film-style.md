## 一、复刻大师镜头

**铁律：** 电影感来自讲清角色、场景和运镜，非堆风格词。

### 五步流程

#### 1. 创建稳定 AI 模特

| ❌ 不要 | ✅ 要 |
|--------|------|
| 急着生成视频 | 先定主角：年龄、气质、服装、妆容、身材、拍摄质感 |
| 单张人物图 | ChatGPT Image 2.0 生成 → Nano Banana Pro 三视图 |

#### 2. 环境服务人物

| ❌ 不要 | ✅ 要 |
|--------|------|
| 随便套场景 | 让 AI 分析模特气质 → 推导匹配的环境氛围 |
| 五镜头五套色调 | 统一色调母版（如：雨后港夜、青绿阴影、琥珀钠灯、35mm 颗粒） |

**示例母版：** 雨后香港夜晚、青绿色阴影、琥珀色钠灯、少量红色霓虹、湿润地面反光、低照度高反差。

各镜头环境差异化但气质统一：
- 希区柯克变焦 → 纵深+监控感空间
- 迈克尔贝英雄 → 雨夜高架桥下停车平台
- 库布里克 → 对称冷峻封闭空间
- 斯皮尔伯格 → 生活感香港后街
- 昆汀后备箱 → 地下停车场/黑色商务车

#### 3. 拆解运镜逻辑（非泛化提示词）

| ❌ 不要 | ✅ 要 |
|--------|------|
| 「帮我写迈克尔贝镜头提示词」 | 找电影片段 → LLM 拆解运镜逻辑 → 补充自己的画面需求 |
| 模糊动作描述 | 明确：人物从哪来？镜头先拍什么？何时移动？停在哪？观众何时看到关键信息？ |

**斯皮尔伯格要点：** 先看环境，再揭示人物。

**迈克尔贝要点：** 明确动作顺序和镜头起点。

#### 4. 整理 Seedance 2.0 视频提示词

五种镜头核心：

| 镜头 | 核心 | 动作逻辑 |
|------|------|----------|
| 希区柯克变焦 | 心理压迫 | 推进+焦距反向变化，人物大小不变、背景拉伸 |
| 迈克尔贝英雄 | 强势登场 | 车入画→开门→低角拍脚→上升环绕→英雄定格 |
| 库布里克 | 空间压迫 | 长焦人物→焦段变短→全景展开更大悬疑空间 |
| 斯皮尔伯格 | 环境叙事 | 跟路人展示街道→路人离开→镜头转向揭示靠墙主角 |
| 昆汀后备箱 | 视角反转 | 后备箱内仰拍→盖开→主角俯视→取物→合盖黑场 |

#### 5. 上传参考生成

人物图 + 环境图 + 推导提示词 → Seedance 2.0。

**锚点：** 同一模特三视图 + 统一港风色调母版。

## 二、《风骨荒原》东方奇幻世界观

**适用：** 概念设计、角色/场景/道具设定、色彩板制作。

### 第一步：AI 发散世界观

| ❌ 不要 | ✅ 要 |
|--------|------|
| 参考作品直接影响世界设定（如双城→上下城工业魔法） | 参考仅作视觉参考，世界观原创 |
| 一次定稿 | 多轮发散 → 锁定关键词 |

**世界一句话：** 远古神兽遗骸化成的荒原上，人类靠骨器、青铜祭术和古道生存；神兽残梦苏醒，主角团寻找共存之道。

**视觉词库：** 东方奇幻、荒原、神兽遗骨、山海经气质、边塞古关、石窟经阁、砂中道观、风沙、青铜器、骨铃、红绳、旧布幡。

**色调：** 骨白、沙金、赭红、烟青、青铜绿、暗朱红。

### 第二步：主角团（每人绑定世界观）

| 角色 | 世界观关联 |
|------|-----------|
| 桑若（拾骨师少女） | 听骨 |
| 沉鸦（哑巴刀客） | 失声 |
| 檀娘（流浪女医师） | 治疗荒原病 |
| 净尘（小和尚） | 超度残梦 |
| 小满（神秘女孩） | 新契约 |

**循环：** 设定 → 出图 → 筛选 → 反馈 → 再生成。

### 第三步：风格统一（材质迁移）

转折点：卡通化面部 + 写实 PBR 服装材质（非纯三渲二）。

| 规则 | 说明 |
|------|------|
| 面部 | 非写实、柔和、干净 |
| 服装道具 | 写实 PBR（旧布、青铜、玉石、骨饰、皮革、木头） |
| 输出 | 白底全身设定图 + 面部/全身三视图 + 服装细节 + 角色道具 |

GPT Image 2.0 以净尘意外材质图为风格参考，迁移全部角色。

### 第四步：场景 + 道具 + 色板

**场景方向：** 国风荒原 + 古关 + 道观 + 石窟 + 茶棚 + 盐湖 + 白草原 + 丹霞峡谷。

**提示词方向：** 风格化 3D 造型 + PBR 写实材质 + 东方奇幻荒原 + 低饱和。

**四类道具板：** 茶棚驿站 / 石窟道观 / 骨风峡谷 / 盐湖古关。

**色彩板 + 材质板：** ChatGPT 一句话生成 → 视频生成时控制画面色彩。

### 第五步：世界观展示页

压缩为一页：世界一句话、关键词、核心元素、主要地域、故事核心、主题表达。

**锚点：** 视觉词库 + 色调六色贯穿人物/场景/道具/色板。

## 三、30 秒日系预告短片

> 溯源：[info-2851](https://www.super-i.cn/info-2851.html)（2026-06 站更）
> 工具：`ChatGPT` · `Google Labs`（Nano Banana）· `seedance2.0` · 可选 FlowPix 一站式

### 七步流程

| 步骤 | 动作 |
|------|------|
| 1 风格 | 参考濱田英明（自然光、低饱和、生活感）→ 见下方三步拆风格 |
| 2 剧本 | 编剧 Skill 丢进 ChatGPT →「启用这个 skill」→ 输入四大板块 → 定为「日系电影预告片」→ 约 30 秒剧本 |
| 3 场景 | **先于人物**：按剧本拆场景（海边路、公交站、旧书、汽水、车票、街道等）→ GPT Image 2.0 **四宫格** → 选最佳 → Nano Banana 2 提取单张 |
| 4 角色 | 脸模+服装参考 → 脸三视图 → 服装三视图 → 服装设计图 → 合成全身 → 脸/半身/全身三视图（男女主各一套） |
| 5 分镜 | 按剧本逐镜：场景 + 人物三视图 + 道具；四宫格选最佳 → 提取单张 prompt |
| 6 视频 | **seedance2.0**：一镜一提示词，不堆复杂长 prompt |
| 7 剪辑 | 按剧本顺序拼接 + BGM + 转场 |

### 第一步拆风格（三步问 LLM）

1. **共性：**「总结参考图在风格与影调上的共性」
2. **不可变条件：**「这些图生成时有哪些不可变条件」
3. **合并：** 将 1+2 结合 → 得到**四大板块**规则（后续场景/分镜/视频全程复用）

**选场景标准：** 构图干净、光线自然、颜色统一、适合放人物、无杂乱元素、日系青春感。

**控制顺序：** 风格 → 剧本 → **场景** → 人物 → 分镜 → 镜头（环境与角色资产都定好再视频化）。

## 五、30 秒高端情绪短片（Codex + 多工具）

> 溯源：[info-2889](https://www.super-i.cn/info-2889.html)（2026-06 站更）
> GCSD 任务写法 → `prompt-director/workflows/agent-task.md`
> 品牌 TVC 管线 → `brand-ad-cg/references/tvc-ads.md`

**适用：** 卫浴、家居、奢侈品等「克制、低曝光、空间质感」情绪片；非卖点堆砌。

### 八步流程

| 步骤 | 动作 |
|------|------|
| 1 概念 | 上传编剧 skill + **参考视频** → Codex 分析镜头/色彩/人物占比/声画节奏 → 选概念方向 |
| 2 分镜迭代 | 30s 分镜 → 升级「导演分镜」（构图/运镜/声音/人物状态）；**写具体可见元素**，非「高级、有情绪」 |
| 3 人设 | 白底定妆（Image 2.0）→ 四视图（Nano Banana Pro）；外国高级感/设计师气质，非网红脸 |
| 4 视觉风格 | 拆参考视频 → 色板 + 风格固定 prompt |
| 5 场景母版 | 先做统一空间（如高端主卫）→ **九宫格多视角** → 后续分镜围绕同一空间 |
| 6 静帧分镜 | 人物镜：人设+场景九宫格+色板；空镜：场景+色板；prompt 结构：**图片标注 + 分镜内容 + 风格固定 + 光线/构图/限制** |
| 7 图生视频 | 普通镜：可灵 o3（树影、水珠、灯带微动）；**转场镜单独**：Seedance 2.0（跨门槛、硬切、方向性动作） |
| 8 剪辑声音 | Codex 排序命名 → BGM 分析参考片 → 跨门槛**声音硬切**；浴室段渐慢渐静；炫光转场 sparingly |

### 分镜写法对比

| ❌ 不要 | ✅ 要 |
|--------|------|
| 城市街边，低曝光，情绪感 | 白天低曝光城市街边，低机位，前景深色车身虚化，中景骑车人暗色下半身，背景暖灰窗格与灰蓝天空 |
| AI 不理解方向时堆 prompt | **手绘草图**标清跨门槛方向 → 图生图纠正 |

### 节奏锚点

- **城市段：** 更碎、更紧，切镜跟 BGM 节奏
- **跨门槛：** 明确切换点（视听同时硬切）
- **空间段：** 渐慢、渐静，水/光/石材承接情绪

## 影视一致性锚点

| 类型 | 锚点 |
|------|------|
| 大师镜头 | 模特三视图 + 统一色调母版 + 运镜动作顺序 |
| 世界观项目 | 视觉词库 + 六色调 + PBR 材质规则 |
| 日系预告 | 四大风格板块 + 角色三视图资产 |
| 高端情绪短片 | 参考片概念 + 场景九宫格 + 转场单独 Seedance |
| 全部 | 参考作品只借形式，不借剧情 |
