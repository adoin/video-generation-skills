## 女频古风短剧：七步工作流

**铁律：** 成片感来自体系统一，非单镜好看。人物、场景、光效、声音、剪辑五维须先锁定。

### 第一步：拆画面质感（先于剧本）

| ❌ 不要 | ✅ 要 |
|--------|------|
| 直接写剧本或生人物 | 先找爆款参考片，截 2 帧让 AI 从摄影师/后期师角度拆解 |
| 写「古风、梦幻、电影感」 | 拆到镜头感、景深、主光方向、柔光、背景虚化、色偏（甜/暖/梦幻） |
| 抄剧情或照搬镜头 | 只提取可复用的画面规则，总结成「画面质感提示词」供全片复用 |

**推荐分析工具：** Gemini / ChatGPT（站内教程用 Gemini 3 Pro 截抖音爆款帧分析）。

**两轮分析 → 一条母版：**
1. 让 AI 描述两帧的**拍摄方式**（镜头感、景深、主光、虚化、色调）
2. 让 AI 把两次分析**合并**为可写进生图/生视频的「画面质感提示词」

**锚点：** 第一步产出的【光线布置】【风格】【拍摄参数】【画面处理】= 全片视觉母版。

### 第二步：人物系统（非单张主角图）

| ❌ 不要 | ✅ 要 |
|--------|------|
| 只生成一张正脸主角图 | 服装三视图 → 服装细节 → 发饰发型 → 面部三视图 → 合成完整参考图 |
| 男主偷懒只做正脸 | 男女主均做胸像三视图 + 全身三视图 |
| 把人物当「一张图」 | 把人物当「角色资产包」，供视频工具多角度理解 |

**锚点：** 服装参考 + 面部参考分开生成再合成；合成图再衍生胸像/全身三视图。

### 第三步：场景九宫格

| ❌ 不要 | ✅ 要 |
|--------|------|
| 每镜重写一个场景 | 先定完整场景设定，再拆同空间多机位 |
| 镜头间环境乱跳 | 用九宫格锁定同一物理空间的多视角 |

**场景 prompt 获取：** 把可填空的场景描述模板发给 LLM（Gemini/ChatGPT），生成完整场景提示词 → 再送 **GPT Image 2.0** 出九宫格。

**锚点：** 场景提示词模板 → GPT Image 2.0 生成九宫格 → 作 seedance2.0 场景参考。

### 第四步：剧本（女频开头抓情绪）

| ❌ 不要 | ✅ 要 |
|--------|------|
| 一上来交代世界观/大段背景 | 前三秒出现人物关系、情绪冲突或暧昧张力 |
| 剧本定稿后才想道具 | 检查开头画面、男女主互动、需提前生成的道具 |
| 关键道具无参考 | 铃铛、河灯等剧本道具 → 三视图，与角色同级资产化 |

**锚点：** 复制站内「剧本 Skill」到 LLM → 启用 Skill → 上传男女主形象 → 要故事框架（非一次定稿）→ 人工校验三问。

### 第五步：分镜拆分（每 15 秒一段）

| ❌ 不要 | ✅ 要 |
|--------|------|
| 整段剧本直接丢进视频工具 | 拆成多个 15 秒段落，段内再拆镜头 |
| 一段塞太多动作 | 15 秒内控制在几个重点镜头 |
| 分镜缺要素 | 每镜写明：景别、构图、光线、表情动作、场景互动、台词、前景背景 |

### 第六步：Seedance 全能参考生成

**提示词结构：**

```
【光线布置】【风格】【拍摄参数】【画面处理】  ← 第一步
【主角形象】                                    ← 人物资产
【分镜画面】                                    ← 第五步
```

| ❌ 不要 | ✅ 要 |
|--------|------|
| 只输入分镜提示词 | 人物图、场景图、道具图、声音参考全部组合 |
| 参考图上传但不标注 | 提示词明确引用：女主图1、男主图2、铃铛图3、河灯图4、场景图5 |
| 跳过声线准备 | 先生成含男女主声音的首段 → 剪映导出男女声线 → 作后续全能参考 |

### 第七步：剪辑统一质感

| ❌ 不要 | ✅ 要 |
|--------|------|
| 直接按顺序硬拼 | 加统一滤镜（如梦幻辉光）弥合段间亮度/对比度/色彩差 |
| BGM 随意选 | 轻梦幻、古风、情绪递进明显的配乐 + 字幕 |

## 童话小剧场：《卖火柴的小女孩》

**适用：** 经典 IP 改编、沉浸式剧场短片、作品集向叙事。

### 前期：风格 + 分镜脚本

| 步骤 | 动作 |
|------|------|
| 1 | 找场景/人物参考图，降低试错成本 |
| 2 | AI 分析参考 → 输出场景英文提示词 |
| 3 | AI 生成故事大纲 + 角色设计 |
| 4 | 链接人物参考 + 大纲 → 角色白底设定图 |
| 5 | 链接场景提示词 + 大纲 → 分幕剧本 |
| 6 | 按幕生成详细分镜英文提示词 |

### 中期：分镜图 → 分段视频

| ❌ 不要 | ✅ 要 |
|--------|------|
| 首帧+尾帧时重复描述场景/风格/角色细节 | 首帧已含视觉信息 → 提示词只写核心动作 + 摄像机运动 |
| 过度指令限制 AI | 给 AI 留创作空间，避免画面变形、动态生硬 |
| 穿帮素材直接用 | 多抽卡筛选；主体移位、元素消失、运镜断层需重生成 |

**首尾帧技巧：** 分镜01 优化亮度 → 首帧图生视频；分镜02 做首帧+尾帧（手触圣诞树）→ 首尾帧生视频。

### 后期：剪映剧场感

1. 按脚本顺序导入片段，删动态不自然部分
2. 转场 + 时长调整保节奏
3. AI 字幕 + 温柔女声配音（或自录，语速匹配画面）
4. 轻柔 BGM + 环境音效（风声、火柴燃烧、街道嘈杂、火苗噼啪）

## 制片一致性锚点（短剧专用）

| 维度 | 锚点 |
|------|------|
| 人物 | 三视图资产包 + 提示词编号引用 |
| 场景 | 九宫格 / 固定空间母版 |
| 光效 | 第一步画面质感提示词贯穿全片 |
| 道具 | 剧本关键道具三视图 |
| 声音 | 首段声线导出作参考 |
| 剪辑 | 统一滤镜 + BGM 情绪线 |

## 表演层：让角色「活起来」（§59）

> 溯源：[info-2880](https://www.super-i.cn/info-2880.html)（2026-06 站更）
> 技法全文 → `prompt-director/references/character-performance.md` §59

在资产与分镜锁定后，**视频 prompt 须安排表演**，非只锁脸与场景。

### 多模态分工（Seedance 2.0）

| 输入 | 锁定 |
|------|------|
| 角色参考图 | 五官、服装 |
| 动作参考视频 | 跑、转身、对峙姿态 |
| 文字 | 镜头顺序、情绪、环境变化 |
| 音频参考 | 环境声、节奏 |

### 单镜：四层动态

身体动作（起终点+速度）→ 跟随物（发、衣、配饰）→ 环境/光线变化 → 声音方向（脚步/人声由远及近）

### 对手戏：调度链

遮挡建立位置 → 观察/触发 → 距离变化（追/退）→ 改变对方行动（挡路/打断）

| ❌ 不要 | ✅ 要 |
|--------|------|
| 第七步后直接拼片 | 关键镜补点缀 SFX；女频加剪映「梦幻辉光」统一段间色调 |
| 两人同框念台词 | 先问：谁先出现、谁发现谁、距离怎么变、谁改变谁 |
