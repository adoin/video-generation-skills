## 制片四维统一

**铁律：** 单张好看 ≠ 成片连贯。五维须先锁定再逐镜生成。

| 维度 | 锚点 | 来源教程 |
|------|------|----------|
| 人物 | 三视图资产包 + 提示词编号引用 | 女频短剧、日系预告、大师镜头 |
| 场景 | 九宫格 / 720 全景 / 俯视图母版 | 女频短剧、虚拟影棚、不穿帮四法 |
| 光效 | 画面质感分析 prompt 贯穿全片 | 女频短剧、企业宣传片影调 |
| 声音 | 首段声线导出作全能参考 | 女频短剧 |
| 剪辑 | 统一滤镜弥合段间色差 | 女频短剧梦幻辉光 |

## 人物一致性

### ❌ 不要

- 只生成一张正脸主角图
- 男女主只做一方三视图
- 视频工具只靠「这个人长什么样」一句描述
- 正脸、侧脸、远景各像一个人

### ✅ 要

**分层资产化：**

```
服装参考 → 服装三视图 → 服装细节
人脸参考 → 面部三视图（正/侧/半侧）
发饰发型提取 → 合成完整人物参考
→ 胸像三视图 + 全身三视图
```

**视频引用：** 提示词明确标注「女主图1、男主图2」等编号。

**3D 漫剧：** 2D 参考 → 标准 3D 三视图资产板（见 narrative/3d-anime-drama.md）。

---

## 场景一致性

### ❌ 不要

- 每镜重写场景 prompt
- 上一秒河边、下一秒庭院、再一秒宫殿
- 让 AI 每个镜头重新想象空间

### ✅ 要

| 方法 | 适用 |
|------|------|
| 场景九宫格 | 短剧同空间多机位 |
| 720° 全景虚拟影棚 | 漫剧自由取景 |
| 俯视图 + 九宫格 | 室内空间逻辑 |
| 360° 环绕视频截图 | 快速多角度背景 |

**原则：** 先有一个完整场景设定，再拆机位（见 scene-continuity.md）。

---

## 道具一致性

| ❌ 不要 | ✅ 要 |
|--------|------|
| 剧本有关键道具但无参考 | 铃铛、河灯等 → 三视图，与角色同级资产 |
| 道具在镜头间变色变形 | 上传道具图并在 prompt 中编号引用 |

## 光效与色调一致性

### 女频短剧

第一步拆参考片 → 【光线布置】【风格】【拍摄参数】【画面处理】= 全片母版。

### 企业宣传片

AI 推导统一企业影调 → 逐张图生图适配（航拍/办公/车间各自微调）。

### 世界观项目（风骨荒原）

六色调色板 + 材质板 → 视频生成时控制色彩。

### 剪辑补救

段间亮度/对比度/色彩仍有差 → 剪辑加统一滤镜（如梦幻辉光）。

## 声音一致性

1. 用提示词生成含男女主声音的首段视频
2. 剪映分别导出男主声线、女主声线
3. 声线文件作 Seedance 全能参考的一部分

## 废片与多镜策略

### 切碎镜头（交叉 prompt-director §24）

| ❌ 不要 | ✅ 要 |
|--------|------|
| 一镜到底 10 秒复杂表演 | 单片段 2–4 秒、一个核心动作 |
| 期望 AI 一次出完整叙事 | 多次生成 → 挑片段 → 剪辑拼合 |

### 废片回收

- 动态不自然部分剪掉
- 穿帮（主体移位、元素消失、运镜断层）→ 重生成或换角度
- 复杂动作：12 宫格分镜 → 多轮筛选正确动作段拼接

### 多角色同框

- 男女主均须完整资产包，否则同框观感明显变差
- 全能参考同时放入多角色设定图 + 场景母版

## 一致性检查清单

生成前确认：

- [ ] 人物三视图资产齐全？
- [ ] 场景有空间母版（九宫格/全景/俯视）？
- [ ] 光效/影调 prompt 已写入每镜？
- [ ] 关键道具有三视图并编号引用？
- [ ] 声线参考已准备？
- [ ] 单段控制在 15 秒内、动作不过载？
- [ ] 剪辑层有统一调色方案？
