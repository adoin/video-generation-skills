## 一、企业宣传片：图片到成片

**铁律：** 画面干净、节奏舒服、信息清楚。样片可全 AI；正式宣传关键画面换真实素材。

### 六步流程

#### 1. 准备图片素材

| 类型 | 来源 | 用途 |
|------|------|------|
| 开场/结尾 | 网络城市航拍（各 1 张） | 建立/收束整体气氛 |
| 中间场景 | AI 生成 | 大楼航拍、大堂、办公、会议、实验室、车间、文化墙、专利墙、签约 |

**建议：** 有条件优先用真实公司素材（大楼、前台、实验室、车间）提升可信度。

#### 2. 统一调色

| ❌ 不要 | ✅ 要 |
|--------|------|
| 不同来源图片直接做视频 | AI 推导统一电影感企业影调 → 图生图逐张处理 |
| 全片套用同一段调色 prompt | 按画面类型微调：航拍强调通透空间感；办公强调自然光商务感；车间强调干净明亮工业感；实验室强调专业整洁 |

**锚点：** 统一影调提示词 = 全片视觉母版（每张图单独适配）。

#### 3. 排列图片 + 推导视频提示词

按宣传片逻辑排序，示例：

```
城市航拍 → 公司大楼 → 大堂 → 办公区 → 会议室 → 实验室 → 车间 → 专利墙 → 签约 → 城市夜景
```

**相邻双图法：** 图1+图2、图2+图3、图3+图4……依次让 AI 推导图生视频提示词。

| ❌ 不要 | ✅ 要 |
|--------|------|
| 所有画面用同一段视频 prompt | 不同场景间镜头运动不同，提示词须贴合画面 |

#### 4. Seedance 2.0 首尾帧生成

```
图1 首帧 + 图2 尾帧 → 段1
图2 首帧 + 图3 尾帧 → 段2
……直至最后一张
```

**优势：** 每段有明确起止画面，衔接可控，比单张图生视频更稳定。

#### 5. 剪映拼接

- 按图片顺序拼接各段视频
- 添加 BGM
- 切换太硬时加简单转场（避免花哨特效）

#### 6. 流程总结

```
开场/结尾图定氛围 → AI 生成中间场景 → 图生图统一色调
→ 相邻双图推导视频 prompt → 首尾帧 Seedance 生成 → 剪映拼接
```

## 二、分镜图故事板（叙事短片）

### 分镜图工作流

| 阶段 | 动作 |
|------|------|
| 文字准备 | 场景分析 → 场景 prompt → 故事大纲 → 角色设定 → 分幕剧本 → 逐幕分镜 prompt |
| 分镜图 | 按幕生成静态分镜 → 亮度/构图优化 |
| 视频化 | 首帧图生视频 / 首尾帧图生视频 |

### 首尾帧故事板规则

| ❌ 不要 | ✅ 要 |
|--------|------|
| 首尾帧时重复描述场景、风格、角色细节 | 首帧已含完整视觉 → 只写核心动作 + 摄像机运动 |
| 过度指令 | 给 AI 创作空间，避免变形和生硬动态 |

**示例：** 「小女孩慢慢伸出手触摸圣诞树，镜头慢慢聚焦到手部」

### 四宫格选图法（日系预告等）

- GPT Image 2.0 四宫格一次出四个版本
- 挑选标准：构图、光线、颜色、人物可置入性、无杂乱元素
- 选中后 Nano Banana 2 单独提取作后续基础

## 三、故事板降低抽卡率

| ❌ 不要 | ✅ 要 |
|--------|------|
| 直接文生视频赌运气 | 先锁定构图和叙事的静态分镜图 |
| 分镜图与视频 prompt 脱节 | 分镜图作 Seedance 首帧/尾帧 + 精简动作 prompt |
| 跳过人工选图 | 四宫格/多版本 → 人工选最佳再视频化 |

**锚点：** 静态分镜图 = 构图+叙事锚；首尾帧 = 运动起止锚。

## 故事板一致性锚点

| 锚点 | 作用 |

| 相邻双图 prompt | 段间运动逻辑 |
| 首帧/尾帧 | 视频衔接控制 |
| 分镜静态图 | 构图叙事预演 |
