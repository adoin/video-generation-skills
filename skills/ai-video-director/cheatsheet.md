# 叙事制片速查

> 两条轨：**Narrative**（短剧/漫剧全流程）与 **Production**（一致性/分镜/故事板）。先闸门定轨。

## 叙事标准流程

```
1. 拆参考片质感（光/调色，先于剧本）
2. 角色设定 + 三视图锚点
3. 场景设定 + 全景/多角度锚点
4. 剧本 → 按镜头拆分镜（非按剧情段）
5. 逐镜：图 → 视频
6. 声音统一 → 剪辑节奏
```

## 制片铁律

| ❌ | ✅ |
|----|-----|
| 先写大段剧情再补镜头 | 每镜一个控制点 |
| 单镜堆所有风格词 | 体系统一：光/色/颗粒一致 |
| 多运镜同句 | 每镜一种运镜或固定 |
| 场景每镜重抽 | 720 全景 / 锚点图 / 标注修正 |
