# AI Video Director

AI **叙事型视频制片**：短剧/漫剧 + 一致性/分镜/故事板。

## 模式

| 模式 | 轨道 | 流程 |
|------|------|------|
| **Narrative** | 短剧/漫剧/短片 | 闸门 → cheatsheet → narrative/ |
| **Production** | 一致性/分镜/动作 | 闸门 → cheatsheet → production/ |

## 创作流水线

```
1. 读 cheatsheet.md
2. 读 confirmation-gates.md — 定轨道与素材
3. 分镜草案 → 用户确认镜号
4. 打开对应 reference（最多 2 个）
5. Prompt → prompt-director
```

## 制片铁律

1. 成片感来自体系统一，非单镜好看
2. 故事板优先
3. 每镜一个控制点
4. 废片回收 → production/consistency
