# TVC / 品牌广告片

> 涵盖：Reactive 扫地机器人 TVC · 宝矿力「蓝色青春」TVC · 比亚迪仰望 U8 汽车 TVC · AI 视频复刻全攻略（TVC 与 UGC）

---

## 标准 TVC 管线

```
创意定调 → 分镜脚本 → 静帧生成（垫图锁一致性）→ 图生视频 → 音效/配乐 → 剪映成片
```

| 阶段 | ✅ 要 | ❌ 不要 |
|------|------|--------|
| 前期 | 先定 15–30s 情绪与卖点，再写分镜 | 直接抽图再凑故事 |
| 静帧 | 全片统一色调；叙事广告用「上一张垫下一张」 | 每张独立生成导致跳色跳脸 |
| 动态 | 首帧 + 尾帧 + 极简运镜词（动作 / 运镜 / 人物状态） | 堆砌长提示词让模型混乱 |
| 后期 | LoFi / Ambient Tech BGM + 环境音 + 产品微音效 | 只有 BGM 无拟音 |

**工具链（按教程）：** Midjourney / Nano Banana → ChatGPT 脚本 → Sora2 / 可灵 / Seedance 2.0 → 剪映

## 模式 A：3C 功能可视化 TVC（扫地机器人）

**适用：** 智能家居、清洁电器、带「透视结构」卖点的 3C

### 分镜结构（约 20s）

| 镜头 | 景别 | 画面重点 | 时长 |
|------|------|----------|------|
| 1 | 远景 | 空间氛围建立，慢推 | ~3s |
| 2 | 中景 | 产品启动，指示灯亮起 | ~3s |
| 3 | 近景 | 避障 HUD / 扫描线特效 | ~4s |
| 4 | 特写 | 清洁动作、粒子吸入 | ~4s |
| 5 | 透视 | 半透明剖视内部结构 | ~3s |
| 6 | 远景 | 空间归于宁静，品牌落版 | ~3s |

### ❌ 不要 / ✅ 要

| ❌ 不要 | ✅ 要 |
|--------|------|
| 6 镜一次生成 | 3 镜一组分批 Sora2 图生视频，降低失控 |
| 脚本只有画面无旁白节奏 | ChatGPT 输出「镜头运动 + 音效 + 旁白」三列分镜 |
| 家居场景过杂 | futuristic minimalist living room + soft natural light |

## 模式 B：叙事情绪 TVC（饮料 / 青春感）

**适用：** 宝矿力类「日漫青春感」品牌片

### 工作流

1. **策划先行** — 故事框架、情绪节奏、运镜逻辑由人定；LLM 仅润色文案
2. **角色锁定** — MJ 半身 → Nano Banana 三视图；主角 + 配角各一套
3. **分镜静帧** — 先统一基础色调；关键镜头用前一张作垫图生成下一张
4. **动态极简** — 静帧完成后，首帧 + 尾帧 + 短运镜词即可
5. **音乐定制** — Suno：`waltz rhythm` + `sweet female vocal` + 夏日清爽标签
6. **剪辑节奏** — 开场慢（车轮特写）→ 缓慢转场 → 海边全景情绪递进

### ❌ 不要 / ✅ 要

| ❌ 不要 | ✅ 要 |
|--------|------|
| 九宫格分镜直接当成片素材 | 九宫格仅作构图参考，产品 / 面部需手修 |
| 动态提示词写满形容词 | 只保留动作、运镜、人物状态 |
| 跳过色调统一 | 全片先定「蓝色青春」色谱再批量出图 |

## 模式 C：汽车极限环境 TVC（仰望 U8）

**适用：** 硬派越野、性能车、需可视化「征服感」

### 工作流

```
品牌概念（无畏 / 掌控）→ 多幕分镜（沙漠 / 雪地 / 涉水…）→ MJ 场景 + 产品垫图 → Seedance 逐镜动态 → 按幕剪辑
```

### 画面原则

| ❌ 不要 | ✅ 要 |
|--------|------|
| 只拍产品特写 | 可视化「征服极端环境」— 车 × 地貌互动 |
| 单场景重复 | 按「幕」切换地貌，每幕 3–4 镜叙事递进 |
| 忽视材质光影 | 复杂光影材质（车漆反光、泥浆、水花）单独精修静帧 |

## 模式 D：爆款 TVC 复刻（换产品）

**适用：** 美妆 TVC、狗粮 UGC 等「借运镜、换主体」

### 通用步骤

1. **拆镜** — 剪映「智能镜头分割」→ 导出每镜首帧
2. **拼长图** — 首帧按播放顺序拼成「分镜长图」（Figma / PS）
3. **换产品** — 分镜长图 + 新产品图 → GPT Image 2.0 批量替换
4. **生成视频** — 音频（≤15s）+ 分镜图 + 产品图 → Seedance 2.0
5. **UGC 加长** — 分镜图拆段（每段 ≤15s）；Gemini 反推结构化镜头脚本；分段生成后剪映拼接

### ❌ 不要 / ✅ 要

| ❌ 不要 | ✅ 要 |
|--------|------|
| 一条提示词跑全长 UGC | 按 12s 前后拆段，每段独立提示词 |
| 替换图有瑕疵直接生成 | 局部重绘圈选修正后再进视频模型 |
| 忽略原片音频节奏 | 先导出原片音频作 Seedance 节奏参考 |

## 模式 E：高端空间情绪 TVC（30s · Codex 工作流）

> 溯源：[info-2889](https://www.super-i.cn/info-2889.html)（2026-06 站更）
> 制片详解 → `ai-video-director/references/narrative/film-style.md` §五
> GCSD 任务 → `prompt-director/workflows/agent-task.md`

**适用：** 卫浴、家居、酒店等「安静、克制、低曝光」高端情绪片；卖空间与感受，非功能清单。

### 概念与分镜

| ❌ 不要 | ✅ 要 |
|--------|------|
| 直接写分镜 | 先上传**参考视频 + 编剧 skill** → 分析镜头/色彩/声画关系 → 选概念 |
| 「画面高级、氛围好」 | 每镜写清前景/中景/背景/构图/光线具体元素 |
| 方向错了继续堆词 | **手绘草图**标跨门槛/动作方向 → 图生图 |

### 资产与生成

1. **人设** — Image 2.0 白底定妆 → Nano Banana 四视图
2. **场景母版** — 统一高端空间描述 → 九宫格（台盆/镜柜/淋浴/地面等多角度）
3. **27+ 静帧** — 模板：图片标注 + 分镜内容 + 风格固定 + 光线/构图/限制
4. **图生视频分流** — 微动空镜：可灵 o3；**转场/硬切动作**：Seedance 2.0

### 声音与节奏

- 城市段：碎、紧，BGM 跟切镜
- **跨门槛：视听硬切**
- 空间段：渐慢渐静；Gemini 音频 ~30s BGM；炫光转场少量使用

## 运镜与提示词速查

| 场景 | 写法片段 |
|------|----------|
| 空间建立 | slow dolly-in, gentle slider movement |
| 产品启动 | top-down push-in, indicator light glow |
| 功能可视化 | wireframe HUD, scanning ripples, x-ray cutaway |
| 叙事衔接 | previous frame as reference, maintain lighting and color grade |
| 图生视频 | [start state] + [camera movement] + [end state] |
