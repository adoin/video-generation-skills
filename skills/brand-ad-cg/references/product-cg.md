## 标准产品 CG 管线

```
产品/草图锚点 → 分镜规划 → 静帧（垫图锁形）→ 首尾帧/图生视频 → 剪映卡点
```

| 阶段 | ✅ 要 | ❌ 不要 |
|------|------|--------|
| 锚点 | 官方产品图 / 白底图 / 构图草图（火柴人即可） | 无参考纯文生图 |
| 静帧 | 简单镜头：产品垫图；复杂透视：草图 + 产品双垫图 | 复杂运镜只靠文字描述 |
| 动态 | 每镜选最合适模型（海螺首尾帧 / 可灵 O1 / Seedance） | 全片单一模型硬套 |
| 后期 | BGM 卡点 + 产品微音效 | 只有音乐无材质音效 |

**常用工具链：** Nano Banana Pro / Artflo → Midjourney → 海螺 / 可灵 / Seedance 2.0 / Liblib → 剪映

## 提示词五维模板（3C / 音箱 / 小家电）

| 维度 | ✅ 要 | ❌ 不要 |
|------|------|--------|
| 主体 | 核心产品 + ≤3 个场景道具 | 道具过多 AI 乱摆 |
| 场景 | 1 主色调 + 暖光/冷光定调 | 五颜六色无主次 |
| 构图 | 中景正视 + 前景轻微遮挡 / 荷兰角特写 | 纯正视呆板或过度倾斜 |
| 色彩 | 主色 70% + 辅色 20% + 点缀 10% | 全饱和撞色 |
| 渲染 | Octane Render / ray tracing / 金属质感自然 | 户外场景加「商业塑料渲染」 |

## 模式 A：草图驱动复杂运镜（手机 CG）

**案例：** 小米 15 Ultra

| 镜头类型 | 方法 |
|----------|------|
| 特写 / 展示 | 产品图单垫图 → 图生图 |
| 穿越镜头 / 大透视 | 草图 + 产品图双垫图 → 图生图 |

**草图原则：** 只需表达构图与机位，不需精细美术

## 模式 B：商业级多场景 CG（音箱）

**案例：** 漫步者 M230

| 场景类型 | 构图要点 |
|----------|----------|
| 中景生活 | 产品居中 + 前景虚化枝叶 |
| 全景客厅 | 广角拉伸空间纵深感 |
| 户外野餐 | 动态道具（外套搭椅）增生活感 |
| 荷兰角桌面 | 高饱和撞色 + 浅景深 |
| 微距特写 | 纯色渐变底 + 金属纹理关键词 |
| 悬浮品牌 | ≤2 个悬浮元素 + logo 对角线构图 |

**进阶：** PS 框选区域 → 提示词局部替换光效（隧道纵深感发光环）

## 模式 C：汽车 CG（SU7 圣托里尼）

```
灵感收集（地貌关键词）→ 3D 白模定机位截图 → 场景先行 AI 建景 → 多视角产品融合 → 视频逐镜
```

| ❌ 不要 | ✅ 要 |
|--------|------|
| 纯 AI 盲盒构图 | Blender 摆机位截白模作透视基础 |
| 忽视车漆与环境的色彩呼应 | 车身色（卡布里蓝）与地中海蓝白场景联动 |
| 现实没有的路硬找素材 | AI「基建」— 虚构环海公路 / 跨海桥段 |

## 模式 D：球鞋悬浮拆解（阿迪达斯）

```
Figma/PS 构图草稿 → 正交前视图 + C4D 风格实验室 → 灰雾电影调色 → 多分镜一致环境 → 拆解动态
```

### 去塑料感三词

`grey atmospheric haze layer` + `low-contrast wash` + `desaturated filmic greyish feel`

### 商业细节

`tiny intricate orange text` + `emissive logo` + `microscopic dials` — 区分练习作与商单

## 模式 H：电商场景与模特（Nano Banana / LTX）

| 模式 | 输入 | 产出 |
|------|------|------|
| 场景适配 | 白底产品 + 目标场景描述 | 多风格场景主图 |
| 虚拟模特 | 服装平铺 / 挂拍 + 模特姿势参考 | 穿搭效果图 |
| LTX 多角度 | 单模特图 | 正面/侧面/背面一致套图 |
| 跨境场景 | 目标市场生活方式关键词 | 本地化场景图 |

### ❌ 不要 / ✅ 要

| ❌ 不要 | ✅ 要 |
|--------|------|
| 模特图面部模糊 | 选清晰正脸 + 干净背景作底图 |
| 场景与产品尺度失调 | 产品图作强参考垫图锁比例 |
