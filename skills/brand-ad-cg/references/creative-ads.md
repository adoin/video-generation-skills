# 创意广告 / MV / 宣传片

> 涵盖：食品/头盔品宣 · 企业宣传片 · 人文科技长片 · 悬疑风城市片 · 3D 跑酷 · 爆炸图一镜到底 · 婚礼动漫 MV · 运动创意 · 宠物 Vlog · 神作结构复刻 · K-pop 舞蹈 MV · 名人创意手机片 · 美妆爆品片

## 模式 A：企业宣传片（图片链式成片）

**适用：** 工厂、科技公司、人文 + 科技调性

### 六步管线

```
素材收集（航拍开场/结尾 + AI 内景）→ 统一调色 → 逻辑排序 → 相邻双图推导运镜词 → Seedance 首尾帧逐段生成 → 剪映拼接 BGM
```

### 画面清单模板

城市航拍 → 公司大楼 → 大堂 → 办公区 → 会议室 → 实验室 → 车间 → 文化/专利墙 → 签约 → 城市夜景

### ❌ 不要 / ✅ 要

| ❌ 不要 | ✅ 要 |
|--------|------|
| 网络图与 AI 图直接混剪 | ChatGPT 推导统一电影感影调词，Nano Banana 逐张图生图调色 |
| 全片同一运镜提示词 | 图 N + 图 N+1 组合，为每对相邻图单独推导过渡运镜 |
| 花哨转场 | 硬切或极简溶解；有真实素材优先实拍 |

## 品牌成长叙事短片（家具工厂 · 2867）

> 溯源：[info-2867](https://www.super-i.cn/info-2867.html)（2026-06 站更）
> 长镜头衔接 → `prompt-director/references/video-prompt.md` §58

**适用：** 工厂/家具/制造业「从小作坊到智能产线」品牌故事，非设备罗列片。

### 六段结构（示例）

| 段落 | 内容 |
|------|------|
| P1 | 年轻创始人进木工作坊，展开椅子图纸 |
| P2 | 画线、锯木、刨削、榫卯 — 第一把椅 |
| P3 | 早期家具厂，机械压制与协作 |
| P4 | **同一木质背板**沿生产线穿越年代（长镜头，可拆三段黑屏转场） |
| P5 | 现代智能工厂，CNC + 视觉检测 |
| P6 | 年迈创始人与年轻工程师交接，椅子进家庭 |

### 管线

```
创意/选题（LLM）→ 手写分镜 → LLM 统一格式（勿擅自加戏）→ 资产清单
→ 人物定妆+三视图 → 电影影调参考（4 片×9 截图分析）→ 场景九宫格
→ 道具（椅子图纸/三视图/背板工序九宫格）→ Seedance 2.0 逐段生成 → 剪辑+BGM
```

| ❌ 不要 | ✅ 要 |
|--------|------|
| AI 分镜直接使用不审 | 逐条查画幅、人物位置、运动方向、设备逻辑 |
| P4 一条视频硬出 | 拆三段，**黑屏切换年代**；镜头绑定同一背板水平运输 |
| 每镜重新发明场景 | 工作坊/家庭用九宫格；P4 草图固定传送带构图再图生图三年代 |

**Seedance prompt 顺序：** 图片标注与用途 → 光线氛围 → 画面内容 → **强制限制词**（如：背板始终水平、同一块背板、生产线横向）

**声音：** 前半钢琴+木质 → 中段机械律动 → 智能段略强 → 结尾温暖弦乐

## 模式 B：150s 人文科技长片

**适用：** 深度品牌叙事、多章节结构

| 章节 | 内容 | 技法 |
|------|------|------|
| 开篇 | 人文情怀 / 城市肌理 | 慢镜 + 环境白噪音 |
| 中段 | 技术实力 / 产线 | 快切 + 数据可视化叠层 |
| 收尾 | 愿景落版 | 航拍拉远 + 品牌 Slogan |

**关键：** 先写章节表再批量出分镜；长片按 15s 单元拆分生成后拼接

## 模式 C：城市美学宣传片（喜鹊谋杀案风）

**适用：** 悬疑感城市文旅、影视化城市形象

```
参考片拉片定调 → MJ 统一冷暖与颗粒 → 分镜按「建立 - 细节 - 情绪」三拍 → 图生视频克制运镜 → 悬疑配乐
```

|--------|------|
| 明亮旅游明信片风 | muted tones, overcast, cinematic grain, British noir atmosphere |

## 模式 E：数码爆炸图一镜到底

**适用：** 3C 拆解、内部结构可视化

### 工作流

1. 产品外观 Hero 静帧
2. 分层拆解静帧（外壳 → 主板 → 芯片…）
3. 即梦 / Seedance 首尾帧：外壳飞散 → 零件悬浮
4. 剪映一镜到底拼接（隐藏跳切点）

| ❌ 不要 | ✅ 要 |
|--------|------|
| 一次生成完整爆炸 | 每层零件独立静帧 + 短段动态 |
| 零件比例随机 | 参考真实爆炸图 / 工程图定层级 |

## 模式 H：K-pop 女团舞蹈 MV（5 步）

1. 参考舞蹈视频拆镜
2. MJ 定角色造型 + 舞台场景
3. 逐镜静帧（垫图锁脸与服装）
4. 图生视频（舞蹈动作词 + 舞台灯光）
5. 剪映按音乐节拍切镜

| ❌ 不要 | ✅ 要 |
|--------|------|
| 单张图直接生全长舞蹈 | 按舞蹈小节拆分，每段 ≤5s |
| 忽略舞台光 | rim light, concert stage, colored spotlights |

## 模式 J：神作复刻（丧尸清道夫）

**适用：** 学习爆款结构、平台流量内容

### 拆解四步

1. **拉片** — 景别、节奏、色调、音效逐镜记录
2. **分镜复刻** — 按原片构图用 AI 重绘（换角色/场景）
3. **动态匹配** — 原片运镜词反推后逐段生成
4. **音效还原** — 环境音 + 冲击音 + BGM 层次对齐原片

| ❌ 不要 | ✅ 要 |
|--------|------|
| 只抄画面不抄节奏 | 剪映对齐原片时间轴再替换画面 |
| 一次生成长镜头 | 按原片镜头切分点拆分生成 |

## 模式 M：头盔 / 硬壳产品场景片（Yema）

```
MJ 多场景静帧（沙漠 / 黄昏 / 云端 / 材质特写）→ 即梦 4.0 定制首尾帧 → MJ 视频动态 → 剪映串联
```

### 提示词结构（产品 CG 通用）

| 维度 | 写法 |
|------|------|
| 主体 | 产品色 + 关键细节（护目镜、纹理） |
| 环境 | 一个主场景 + 不超过 3 个道具 |
| 风格 | photorealistic / cinematic lighting |
| 动态 | heat haze / sand impact / airflow from vents |
