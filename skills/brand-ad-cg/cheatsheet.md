# 品牌广告 / CG 速查

> 顺序：**确认闸门** → 定视觉策略 → 按类型打开 **1–2 个** reference。

## 与电商的边界

| 维度 | brand-ad-cg | ecommerce |
|------|-------------|-----------|
| 目标 | 品牌势能、审美标杆 | 转化、铺量、详情页 |
| 节奏 | 电影感、留白、仪式感 | 信息密度、卖点直给 |
| 时长 | 15s–60s TVC / 概念片 | 15–30s 种草切片 |

## 标准 CG 链路

```
创意方向 / 品牌调性
  → 风格板 / 参考片分析（光、镜头、调色）
  → 关键帧 / 分镜（可借 ai-video-director 故事板）
  → 产品/场景生成
  → 图生视频片段
  → 剪辑节奏 + 调色统一
```

## 视觉策略三件套

| 要素 | 必答 |
|------|------|
| 光 | 主光方向、光比、是否有轮廓光/体积光 |
| 镜头 | 焦段感、运镜种类（每段最多一种） |
| 色 | 主色 + 辅助色；是否参考具体品牌片 |
